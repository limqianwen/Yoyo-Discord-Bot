# Yoyo - AI-Powered Discord Study Assistant 🤖

Yoyo is an AI-powered Discord assistant designed to support students throughout their academic journey by helping them learn, organize their studies, and manage academic tasks efficiently.

## ✨ Features

### 🤖 AI Learning Tutor
- Ask Yoyo questions and receive AI-powered explanations.

### 📄 Smart Study Materials
- Generate summaries for any topic.
- Create flashcards for effective revision.
- Generate quizzes to test your understanding.

### 📅 Assignment & Study Planner
- Manage assignments and study tasks.
- Track completed and missed tasks.

### 📊 Personal Dashboard
- View your learning statistics and activity progress.

### 🔔 Smart Notifications
- Receive reminders for upcoming deadlines.

---

## 📌 Commands

**Prefix:** `!`

### 🤖 AI Learning Tutor
| Command | Description |
|---------|-------------|
| `!ask <question>` | Ask Yoyo anything! |

### 📄 Study Materials
| Command | Description |
|---------|-------------|
| `!summary <topic>` | Generates a summary for a topic. |
| `!flashcard <topic>` | Generates 5 flashcards. |
| `!quiz <topic>` | Generates a 5-question quiz. |

### 📅 Assignment & Study Planner
| Command | Description |
|---------|-------------|
| `!viewtask` | View your tasks. |
| `!addtask` | Add a task. |
| `!deltask <task ID>` | Delete a task. |
| `!check <task ID>` | Mark task as done. |
| `!missed` | View missed tasks. |

### 📊 Dashboard
| Command | Description |
|---------|-------------|
| `!profile` | View your profile. |

---

## 🎯 Project Goal

Yoyo aims to make learning more accessible by integrating AI assistance, study tools, and task management into a familiar Discord environment, allowing students who frequently use Discord to learn, organize, and manage their academic journey more efficiently.
